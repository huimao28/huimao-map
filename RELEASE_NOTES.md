# 灰猫地图 1.0.17 更新说明

## 修复内容

- 修复 Android Auto 地图旋转时屏幕四角或瓦片边缘频繁出现黑块的问题。
- Android Auto 地图改为稳定的北向上显示，不再旋转整张瓦片地图；车辆箭头根据实际航向旋转，仍可识别行驶方向。
- 瓦片预取由屏幕对角线大范围缓存改为屏幕四周一圈（256 像素），减少同进程下载、解码和 Canvas 绘制压力，降低对手机百度导航渲染的影响。

## 未修改内容

- 未修改手机百度导航 View、主题、窗口布局、路线规划和导航生命周期。
- 未修改手机与 Android Auto 定位、路线坐标系自动判断及惯性导航。
- 未修改 Android Auto 左上角指引同步。
- 三个微信位置转发插件仅随版本重新构建，转发逻辑未修改。

## 已知问题

- Android Auto 地图现在固定北向上，不再保持车辆箭头始终朝屏幕上方。
- 网络不可用或百度瓦片服务器响应失败时，尚未下载的区域仍可能短暂显示深色背景。
- 手机导航顶部安全区域在部分设备上仍可能与百度面板颜色不完全一致。

## 安装说明

- 主应用 `versionCode` 为 18，使用固定长期 release 证书，可直接覆盖安装 1.0.16。
- Release 同时提供百度、高德和腾讯三个微信位置转发插件；只需安装一个与现有地图应用不冲突的版本。
