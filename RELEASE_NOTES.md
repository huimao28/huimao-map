# 灰猫地图 1.0.14 更新说明

## 修复内容

- 修复 Android Auto 左上角路线指引与手机百度导航当前指引不一致的问题。
- 百度导航回调仍作为即时兜底；回调后延迟 100 毫秒，只读手机端当前真正可见的距离和道路指引控件，再同步到 Android Auto。
- 同名但隐藏、透明、尺寸为零或未显示的百度导航控件不会参与同步，避免旧面板文字覆盖当前指引。

## 未修改内容

- 未修改百度路线规划、导航启动和退出生命周期。
- 未修改手机端百度导航 View、主题、全屏或窗口布局。
- 未修改定位源、坐标转换、惯性导航和 Android Auto 地图瓦片绘制。
- 三个微信位置转发插件仅随版本重新构建，转发逻辑未修改。

## 已知问题

- Android Auto 定位仍可能存在视觉偏移。
- 手机导航顶部仍有状态栏留白。
- Android Auto 地图旋转或瓦片未及时下载时可能出现黑块。
- 不同百度导航面板模式下，若 SDK 不暴露可见文字控件，Android Auto 会回退使用百度导航回调文字。

## 安装说明

- 主应用 `versionCode` 为 15，使用固定长期 release 证书，可直接覆盖安装 1.0.13。
- Release 同时提供百度、高德和腾讯三个微信位置转发插件；只需安装一个与现有地图应用不冲突的版本。
