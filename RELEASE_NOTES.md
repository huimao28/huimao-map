# 灰猫地图 1.0.20 更新说明

## 修复内容

- 修复 1.0.15 引入的 Android Auto 车辆位置与蓝色路线整体偏移：撤销根据单次定位自动执行 GCJ-02 → BD-09 路线转换，恢复直接使用百度导航 SDK 返回的原始路线坐标。
- 修复 50 米比例尺长度计算错误：不再使用标准 Web Mercator 米/像素公式，改为在百度墨卡托坐标中计算当前位置向东 50 米两点的实际像素距离。
- 地图缩放从 20 恢复为 18，避免视野过近；比例尺仍保持真实 50 米。

## 未修改内容

- 未修改手机百度导航 View、路线规划、导航生命周期和手机定位。
- 未修改 Android Auto 实时定位、惯性导航和指引同步。
- 未修改北向上地图模式与瓦片下载策略。
- 三个微信位置转发插件仅随版本重新构建，转发逻辑未修改。

## 已知问题

- 百度导航 SDK 在其他版本或设备若返回不同坐标系，本版不会自动转换，优先保证当前实际设备路线与定位一致。
- 网络不可用或瓦片尚未下载时，局部地图仍可能暂时显示深色背景。
- 手机导航顶部安全区域在部分设备上仍可能与百度面板颜色不完全一致。

## 安装说明

- 主应用 `versionCode` 为 21，使用固定长期 release 证书，可直接覆盖安装 1.0.19。
- Release 同时提供百度、高德和腾讯三个微信位置转发插件；只需安装一个与现有地图应用不冲突的版本。
