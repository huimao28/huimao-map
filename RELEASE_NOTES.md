# 灰猫地图 1.0.15 更新说明

## 修复内容

- 修复 Android Auto 中车辆位置与蓝色导航路线可能出现整体视觉偏移的问题。
- 百度导航适配层在不同版本中可能输出 GCJ-02 或 BD-09 路线坐标；同步路线到 Android Auto 时，会分别计算原始路线和 GCJ-02 转 BD-09 后路线与当前 BD-09 定位的距离。
- 仅当转换后至少改善 25 米且距离降低 30% 以上时采用转换结果，避免对本来已经是 BD-09 的路线重复转换。
- 增加坐标判断日志，记录原始距离、转换后距离和最终选择，便于后续定位具体设备差异。

## 未修改内容

- 未修改手机定位客户端、百度导航引擎外部定位输入和定位质量过滤。
- 未修改路线规划、导航启动/退出生命周期及手机百度导航 View。
- 未增加路线吸附或强制修改车辆实时位置。
- 未修改 Android Auto 指引同步和地图瓦片绘制。
- 三个微信位置转发插件仅随版本重新构建，转发逻辑未修改。

## 已知问题

- 手机导航顶部仍有状态栏留白。
- Android Auto 地图旋转或瓦片未及时下载时可能出现黑块。
- 路线起点离当前定位较远（例如刚发生路线重算）时，坐标系自动判断会保守地保留原始路线。

## 安装说明

- 主应用 `versionCode` 为 16，使用固定长期 release 证书，可直接覆盖安装 1.0.14。
- Release 同时提供百度、高德和腾讯三个微信位置转发插件；只需安装一个与现有地图应用不冲突的版本。
