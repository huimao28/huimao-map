# 灰猫地图 1.0.23 更新说明

## 修复内容

- 继续修复手机百度导航顶部安全区域显示白色空条的问题。
- 前一方案只设置 Activity 窗口背景，实际白色区域来自百度导航 SDK 返回的根 View，因此未生效。
- 本版只将百度导航根 View 的默认背景设置为深色 `#1F2937`，用于填充地图和顶部指引没有覆盖的安全区域。

## 未修改内容

- 未隐藏系统栏，未启用新的全屏方案，未调整 WindowInsets。
- 未改变百度导航 View 的尺寸、位置和任何子控件布局。
- 未修改路线规划、导航生命周期、定位、Android Auto 地图、比例尺和指引同步。
- 三个微信位置转发插件仅随版本重新构建，转发逻辑未修改。

## 已知问题

- 如果白色区域来自百度 SDK 内部某个具体子容器而非根 View，本版可能只能改变部分空白区域；后续需根据新截图定位该容器。
- Android Auto 车辆与路线显示校准、50 米比例尺在部分车机上仍可能不符合预期，本版未继续修改。

## 安装说明

- 主应用 `versionCode` 为 24，使用固定长期 release 证书，可直接覆盖安装 1.0.22。
- 百度 AK 应绑定包名 `com.huimao.map` 和 Release SHA-1 `DE:FF:00:C2:E1:8A:20:62:9C:4A:17:67:B7:27:A5:08:CC:1B:F0:E4`。
- Release 同时提供百度、高德和腾讯三个微信位置转发插件。
