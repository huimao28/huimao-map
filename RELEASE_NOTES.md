# 灰猫地图 1.0.19 更新说明

## 修复内容

- 根据实际车机截图继续放大 Android Auto 地图显示比例。
- 地图缩放级别由 18 提高到 20，车辆周围道路和路口显示更大，更适合驾驶时快速识别。
- 50 米比例尺继续使用真实地理距离动态计算，不因放大地图而虚标。

## 未修改内容

- 未修改手机百度导航 View、路线规划、导航生命周期和手机定位。
- 未修改 Android Auto 定位、路线坐标系判断、惯性导航和指引同步。
- 未修改北向上地图模式和瓦片下载逻辑，仅提高地图缩放级别。
- 三个微信位置转发插件仅随版本重新构建，转发逻辑未修改。

## 已知问题

- 缩放级别 20 下，车机视野更近，较远处的路线不会同时显示在屏幕内。
- 网络不可用或瓦片尚未下载时，局部区域仍可能暂时显示深色背景。

## 安装说明

- 主应用 `versionCode` 为 20，使用固定长期 release 证书，可直接覆盖安装 1.0.18。
- Release 同时提供百度、高德和腾讯三个微信位置转发插件；只需安装一个与现有地图应用不冲突的版本。
