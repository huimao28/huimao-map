# 灰猫地图 1.0.18 更新说明

## 修复内容

- 放大 Android Auto 自绘地图显示比例，从缩放级别 15 提高到 18，减少开车时地图过小、道路信息拥挤的问题。
- 增加真实的 50 米比例尺，比例尺长度根据当前纬度和缩放级别动态计算，不是固定装饰线。
- 比例尺显示在地图左下角，使用白色描边和阴影，保证在百度底图上可读。

## 未修改内容

- 未修改手机百度导航 View、路线规划、导航生命周期和手机定位。
- 未修改 Android Auto 定位、路线坐标系判断、惯性导航和指引同步。
- 未改变北向上地图模式和瓦片下载策略，只调整车机地图缩放级别。
- 三个微信位置转发插件仅随版本重新构建，转发逻辑未修改。

## 已知问题

- Android Auto 地图仍固定北向上，车辆箭头随航向旋转。
- 网络不可用或瓦片尚未下载时，局部地图仍可能暂时显示深色背景。
- 50 米比例尺在极高或极低纬度地区会按当前纬度换算，视觉长度会不同。

## 安装说明

- 主应用 `versionCode` 为 19，使用固定长期 release 证书，可直接覆盖安装 1.0.17。
- Release 同时提供百度、高德和腾讯三个微信位置转发插件；只需安装一个与现有地图应用不冲突的版本。
