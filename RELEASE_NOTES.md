# 当前版本更新说明

## 修复内容

- 将 GitHub Release 更新说明纳入仓库维护，每次发布必须明确列出改动内容。
- GitHub Actions 发布前自动检查更新说明，缺失内容或仍含“待填写”时停止发布。
- Release 页面直接使用本文件作为正文，不再仅显示自动生成的提交记录。

## 未修改内容

- 本次只修改发布流程和文档，没有修改主应用、百度导航、Android Auto 地图或三个转发插件的运行逻辑。

## 已知问题

- Android Auto 左上角指引可能与手机当前指引不一致。
- Android Auto 定位仍可能存在视觉偏移。
- 手机导航顶部仍有状态栏留白。
- Android Auto 地图旋转或瓦片未及时下载时可能出现黑块。

## 安装说明

- 本次仅建立发布说明机制，不发布新的 APK 版本。
- 主应用和三个转发插件继续使用固定长期 release 证书。
