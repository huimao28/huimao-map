# 灰猫地图 1.0.22 更新说明

## 修复内容

- 根据实际车机截图修复 Android Auto 车辆箭头与蓝色路线之间的显示偏移。
- 不再转换或修改百度原始路线坐标；仅在 Android Auto 绘制层中查找离当前车辆最近的路线点，并对整条路线应用同一个平移量，使车辆与当前路线对齐。
- 显示平移限制为 300 像素以内；超过限制时不校准，避免把路线移动到错误道路。
- Android Auto 地图缩放由 18 提高到 20，使真实 50 米比例尺在车机宽屏上具有更明显的显示长度。
- 50 米比例尺继续使用百度墨卡托两点像素差计算，最小可读长度由 80 像素提高到 120 像素。

## 未修改内容

- 未修改百度路线规划、原始路线数据、手机定位和导航生命周期。
- 未修改手机百度导航 View 和 Android Auto 指引同步。
- 未修改瓦片来源、北向上模式和惯性导航逻辑。
- 三个微信位置转发插件仅随版本重新构建，转发逻辑未修改。

## 已知问题

- 显示层校准只处理车辆与路线的整体平移，不处理路线几何本身的局部误差。
- 缩放级别 20 下视野更近，较远路线不会同时显示在屏幕内。
- 网络不可用或瓦片尚未下载时，局部地图仍可能暂时显示深色背景。

## 安装说明

- 主应用 `versionCode` 为 23，使用固定长期 release 证书，可直接覆盖安装 1.0.21。
- 百度 AK 应绑定包名 `com.huimao.map` 和 Release SHA-1 `DE:FF:00:C2:E1:8A:20:62:9C:4A:17:67:B7:27:A5:08:CC:1B:F0:E4`。
- Release 同时提供百度、高德和腾讯三个微信位置转发插件。
