# 灰猫地图 1.0.16 更新说明

## 修复内容

- 修复手机百度导航顶部安全区域显示为突兀白条的问题。
- 不再尝试隐藏系统栏或改变百度导航 View 尺寸，仅将导航 Activity 的窗口背景和状态栏区域设置为与百度顶部指引面板协调的深色。
- 日间使用 `#1F2937`，夜间使用 `#16191D`，状态栏图标保持浅色，避免白底白条和图标看不清。

## 未修改内容

- 未修改百度导航 View、WindowInsets、全屏标志、路线规划及导航生命周期。
- 未修改手机和 Android Auto 的定位、路线坐标判断及惯性导航。
- 未修改 Android Auto 指引同步和地图瓦片绘制。
- 三个微信位置转发插件仅随版本重新构建，转发逻辑未修改。

## 已知问题

- Android Auto 地图旋转或瓦片未及时下载时可能出现黑块。
- 部分厂商系统可能强制覆盖状态栏颜色；这种情况下顶部区域会由系统主题决定，但不会改变百度导航内容布局。

## 安装说明

- 主应用 `versionCode` 为 17，使用固定长期 release 证书，可直接覆盖安装 1.0.15。
- Release 同时提供百度、高德和腾讯三个微信位置转发插件；只需安装一个与现有地图应用不冲突的版本。
